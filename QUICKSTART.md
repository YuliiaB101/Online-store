# Быстрый старт

## 1. Настройка базы данных PostgreSQL

Убедитесь, что PostgreSQL установлен и запущен, затем выполните:

```bash
# Войдите в PostgreSQL
psql -U postgres

# Выполните SQL скрипт для создания базы данных и таблиц
\i backend/config/database.sql

# Или на Windows:
psql -U postgres -f backend/config/database.sql
```

## 2. Запуск Backend

```bash
cd backend
yarn dev
```

Backend будет доступен на http://localhost:5000

## 3. Запуск Frontend

Откройте новый терминал:

```bash
cd frontend
yarn start
```

Frontend будет доступен на http://localhost:3000

## Готово!

Теперь вы можете:
- Зарегистрироваться в системе
- Просматривать каталог товаров
- Использовать фильтры и поиск
- Добавлять товары в избранное
- Добавлять товары в корзину
- Лайкать товары
